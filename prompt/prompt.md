# Prompt合集
## 提取文献相关
### 给文献命名
```
请根据我提供的文献信息生成标准文件名，格式为：年份_通讯作者姓氏_文献核心标识。具体规则如下：

年份提取：
优先使用DOI中的出版年份（若有）
若无DOI则采用期刊标注的接受年份
格式保持4位数字，如2023
通讯作者处理：
提取标注*或corresponding author的姓氏
多位通讯作者时取首位
若无标注则取最后一位作者
保留姓氏原始大小写格式
文献标识生成：
优先使用原标题中的核心短语（3-5个关键词）
若标题过长则生成英文缩写（不超过8个字母）
保留原标题的重要专业术语
处理要求：
• 删除所有特殊符号（&,/,:等）
• 替换空格为下划线
• 英文单词首字母大写
• 总长度控制在35字符内
示例：
输入：Wang et al. (2022) "Novel CRISPR-Cas9 applications in neurodegenerative disease modeling" DOI: 10.1038/s41586-022-05533-z (*通讯作者：ZHANG Wei)
输出：2022_Zhang_CRISPR_Cas9_Neurodegenerative
```
